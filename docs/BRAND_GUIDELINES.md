# IncomeProof Brand Guidelines

**Version:** 1.0
**Status:** Approved

---

# Purpose

This document defines the visual identity of IncomeProof.

Its purpose is to ensure a consistent experience across:

* Web application
* Landing page
* Documentation
* Marketing materials
* Pitch decks
* Future mobile applications

Every designer, developer, and AI coding assistant should follow these guidelines.

---

# Brand Positioning

IncomeProof is a **trust infrastructure platform**.

The product should communicate:

* Trust
* Professionalism
* Transparency
* Simplicity
* Security
* Financial credibility

It should **not** look like:

* A cryptocurrency exchange
* A DeFi dashboard
* An NFT marketplace
* A gaming application

The design language should resemble:

* Stripe
* Linear
* Notion
* Mercury
* Modern banking dashboards

---

# Brand Personality

IncomeProof should feel:

* Calm
* Professional
* Modern
* Reliable
* Intelligent
* Minimal

Never flashy.

Never overwhelming.

Never "crypto".

---

# Logo

For MVP:

Use a simple wordmark.

```
IncomeProof
```

Typography should be clean and modern.

Future versions may include an icon representing:

* Trust
* Verification
* Connected credentials
* Digital proof

Avoid generic blockchain cubes or cryptocurrency symbols.

# Spacing

Use a consistent spacing scale.

Preferred spacing:

* 4
* 8
* 12
* 16
* 24
* 32
* 48
* 64

Avoid arbitrary spacing values.

---

# Border Radius

Cards

Medium rounded corners

Buttons

Medium radius

Inputs

Medium radius

Avoid excessive rounding.

The interface should feel professional rather than playful.

---

# Shadows

Use subtle elevation.

Cards:

Soft shadow

Dialogs:

Medium shadow

Avoid dramatic shadows or glowing effects.

"Verification is temporarily unavailable."

---

# Product Voice

Interface text should be:

Professional

Clear

Concise

Helpful

Avoid blockchain jargon.

Say:

"Verification Complete"

Instead of:

"Transaction Successfully Confirmed On-Chain"

Focus on outcomes users understand.

---

# Design Principles

Every screen should answer three questions immediately:

1. What am I looking at?
2. What should I do next?
3. Can I trust this information?

If the answer to any of these is unclear, simplify the interface.

---

# AI Design Rules

When generating UI:

Prioritize:

* Simplicity
* Readability
* Consistency
* Accessibility
* Trust

Do not:

* Invent additional themes
* Use random accent colors
* Create crypto-style dashboards
* Overuse gradients
* Add unnecessary visual effects

---

# Brand Promise

IncomeProof helps people prove verified income through a secure, trustworthy, and privacy-preserving experience.

Every design decision should reinforce confidence and clarity.
